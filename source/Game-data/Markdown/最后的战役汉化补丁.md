## 使用方法

在Steam库存页右键游戏名->管理->浏览本地文件打开游戏根目录，然后进入The Last Stand Trilogy\TheLastStandLegacyCollection.app文件夹，将汉化包内的文件拖入替换即可。

![](/images/最后的战役汉化补丁/使用说明.webp)
